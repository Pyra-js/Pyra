> A full-stack web framework for blazing-fast web development.

Pyra.js is a TypeScript-first framework for building modern full-stack web applications. File-based routing, server-side rendering, an HMR dev server, esbuild-powered production builds, and interactive project scaffolding, all out of the box.

> **Pre-release software.** Pyra.js is under active development and has not reached a stable v1.0 release. APIs, configuration, and behavior may change between versions. Some features may not work as intended. If you run into a bug, please [open an issue](https://github.com/Natejsx/Pyra/issues). If you discover a security vulnerability, please [report it responsibly](https://github.com/Natejsx/Pyra/security).
